### bonanza-mcp-guard / [mcp-guard](https://github.com/c6zks4gssn-droid/mcp-guard)

Lightweight **stdio MCP gateway** for Claude Desktop, Cursor, and custom agents.

- API key / JWT auth, per-agent rate limits, spend caps on wallet/x402 tools, JSONL audit
- `mcp-guard scan` — audit local MCP client configs

```bash
pip install "bonanza-mcp-guard[yaml]"
```

PyPI: https://pypi.org/project/bonanza-mcp-guard/ · Apache-2.0

Complements [mcp-scan](https://github.com/invariantlabs-ai/mcp-scan) with a runtime proxy.
